# DeskQuiver

**Launch all your apps and windows with one click.**

DeskQuiver is a Windows workspace manager that lets you define groups of apps, browser tabs, and terminal sessions — then launch everything at once with a single switch.

## Download

→ **[Latest release](https://github.com/Sajinthan/desk-quiver-releases/releases/latest)**

## Requirements

- Windows 10 or Windows 11 (64-bit)

## Installation

1. Download and run the installer
2. DeskQuiver starts in the system tray
3. Use the tray icon or your configured hotkey to open the app
